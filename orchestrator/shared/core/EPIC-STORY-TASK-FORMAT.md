## EPIC.STORY.TASK FORMAT REQUIREMENTS

- **CRITICAL** use [0-9]{4} format for EPIC ID, zero-padded (e.g. 0001, 0002, 0003)
- **CRITICAL** use [0-9]{2} format for STORY ID, zero-padded (e.g. 01, 02, 03)
- **CRITICAL** use [0-9]{2} format for TASK ID, zero-padded (e.g. 01, 02, 03)
- **CRITICAL** use ONLY {EPIC_ID}(.{STORY_ID}(.{TASK_ID})?)? pattern
- **CRITICAL** maintain sequential numbering within each level
- **CRITICAL** only use digits and dots, no letters or other symbols
- **CRITICAL** no suffixes and no prefixes
