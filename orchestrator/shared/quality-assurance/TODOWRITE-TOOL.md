## TODOWRITE TOOL

### Todo Structure
- **Analyze requirements** examining test specifications and quality standards
- **Assess test coverage** reviewing existing tests and coverage metrics
- **Execute validation** performing comprehensive quality checks
- **Define quality gates** establishing pass/fail criteria
- **Link test results** building complete validation documentation
- **Validate thoroughly** performing final quality assessment

### Requirements
- **Track validation progression** through todo state transitions
- **Maintain visibility** of current testing across all phases
- **Update immediately** when validation transitions occur
- **Coordinate handoffs** between different validation activities
- **Monitor completion** ensuring all tests reach completed state
