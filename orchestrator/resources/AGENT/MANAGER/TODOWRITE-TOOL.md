## TODOWRITE TOOL

### Todo Structure
- **Analyze context** examining requirements and technical specifications
- **Assess architecture** reviewing existing system structure and patterns
- **Execute planning** creating comprehensive technical specifications
- **Define validation** establishing comprehensive acceptance criteria
- **Link documentation** building complete documentation hierarchy
- **Validate technically** performing final validation and acceptance

### Requirements
- **Track activity progression** through todo state transitions
- **Maintain visibility** of current work across all activities
- **Update immediately** when activity transitions occur
- **Coordinate handoffs** between different agent activities
- **Monitor completion** ensuring all todos reach completed state
